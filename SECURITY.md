# Security Policy

## Supported Versions
We support the latest `v0.x` release line on a best-effort basis.

## Reporting a Vulnerability
Please do NOT open a public issue for security reports.

Instead, use GitHub Security Advisories for this repository:
https://github.com/pas7studio/bun-ready/security/advisories

If you cannot use GitHub advisories, you may contact Pas7 Studio via:
https://pas7.com.ua/
